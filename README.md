# SILENt

## Install
### Debian/Ubuntu

```bash
sudo apt install git -y &&
git clone https://github.com/kotik06/SILENt &&
cd SILENt &&
chmod +x install.sh &&
./install.sh
```
### Arch 

```bash
sudo pacman -S git --noconfirm&&
git clone https://github.com/kotik06/SILENt &&
cd SILENt &&
chmod +x install.sh &&
./install.sh 
```
### Termux 

```bash
pkg install git -y &&
git clone https://github.com/kotik06/SILENt &&
cd SILENt &&
chmod +x install.sh &&
./install.sh
```
