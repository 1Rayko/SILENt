# SILENt

## Install
### Debian/Ubuntu

```bash
sudo apt install git -y &&
git clone https://github.com/kotik06/SILENt &&
cd SILENt &&
chmod +x install.sh &&
./install.sh &&
python3 go.py
```
### Arch 

```bash
sudo pacman -S git --noconfirm&&
git clone https://github.com/kotik06/SILENt &&
cd SILENt &&
chmod +x install.sh &&
./install.sh &&
python3 go.py
```

```bash
pkg install git -y &&
git clone https://github.com/kotik06/SILENt &&
cd SILENt &&
```
